# Useful Keybindings
-Full cheat sheet https://tmuxcheatsheet.com/

## Plugins
-mod + I = install plugins

## Sessions
- tmux a = attach to last session
- tmux a -t x = attach to session x
- mod w = session and window preview
- mod ( = prev session
- mod ) = next session

## Windows
- mod c = new window
- mod n = next window
- mod w = prev window

## Panes
- mod | = horiz split
- mod - = vert split
- mod ! = convert pane to window
